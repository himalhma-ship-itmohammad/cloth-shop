HTML টিউটোরিয়াল

HTML কি?

HTML (HyperText Markup Language) হল ওয়েব পেজ তৈরি করার জন্য স্ট্যান্ডার্ড মার্কআপ ভাষা। এটি ওয়েব ব্রাউজারে কন্টেন্ট কাঠামো এবং উপস্থাপনা বর্ণনা করে।

<!DOCTYPE html>
<html>
<head>
  <title>পেজ টাইটেল</title>
</head>
<body>
  <h1>আমার প্রথম হেডিং</h1>
  <p>আমার প্রথম প্যারাগ্রাফ।</p>
</body>
</html>

CSS মার্জিন (Margin)

মার্জিন প্রপার্টি একটি এলিমেন্টের বাইরের স্পেস নির্ধারণ করে। এটি এলিমেন্টের চারপাশের স্পেস তৈরি করে।

selector {
  margin: value;
}

মানসমূহ:

auto - ব্রাউজার স্বয়ংক্রিয়ভাবে মার্জিন গণনা করে
length - px, pt, cm, ইত্যাদিতে মার্জিন নির্ধারণ করে
% - ধারক এলিমেন্টের প্রস্থের % এ মার্জিন নির্ধারণ করে
inherit - মার্জিন প্যারেন্ট এলিমেন্ট থেকে উত্তরাধিকার সূত্রে প্রাপ্ত
মার্জিন উদাহরণ

CSS কোড:

.box {
  margin: 20px;
  padding: 15px;
  background: #3498db;
  color: white;
  border-radius: 5px;
}

প্রিভিউ:

এই বক্সটির 20px মার্জিন আছে

CSS সাধারণ প্রপার্টিসমূহ

প্রপার্টি বর্ণনা মান
color টেক্সটের রং নির্ধারণ করে red, #FF0000, rgb(255,0,0)
background-color ব্যাকগ্রাউন্ড রং নির্ধারণ করে blue, #0000FF, rgba(0,0,255,0.5)
font-size টেক্সটের সাইজ নির্ধারণ করে 16px, 1em, 100%
margin এলিমেন্টের বাইরের স্পেস নির্ধারণ করে 10px, 5%, auto
padding এলিমেন্টের ভিতরের স্পেস নির্ধারণ করে 15px, 2em, 10%
border বর্ডার স্টাইল, প্রস্থ এবং রং নির্ধারণ করে 1px solid black

CSS টিউটোরিয়াল

CSS সিনট্যাক্স

CSS (Cascading Style Sheets) হল একটি স্টাইলশীট ভাষা যা HTML ডকুমেন্টের উপস্থাপনা বর্ণনা করে।

selector {
  property: value;
  property: value;
}
নোট:

CSS সিলেক্টর HTML এলিমেন্ট নির্বাচন করতে ব্যবহৃত হয় যেগুলো আপনি স্টাইল করতে চান।

CSS উদাহরণ

HTML:

<div class="container">
  <h1>শিরোনাম</h1>
  <p>একটি প্যারাগ্রাফ</p>
</div>

CSS:

.container {
  background: #f0f0f0;
  padding: 20px;
  border-radius: 8px;
}

h1 {
  color: #2c3e50;
  text-align: center;
}

p {
  color: #7f8c8d;
  line-height: 1.6;
}

JavaScript পরিচিতি

JavaScript হল একটি প্রোগ্রামিং ভাষা যা ওয়েব পেজগুলিকে ইন্টারেক্টিভ করে তোলে। এটি একটি স্ক্রিপ্টিং ভাষা যা ক্লায়েন্ট-সাইডে চলে।

// ভেরিয়েবল ডিক্লেয়ার
let name = "John";
const age = 30;

// ফাংশন ডিফিনিশন
function greet(person) {
  return "Hello, " + person;
}

// ফাংশন কল
console.log(greet(name));
JavaScript উদাহরণ

JavaScript কোড:

// DOM ম্যানিপুলেশন
const button = document.getElementById('myButton');
const message = document.getElementById('message');

button.addEventListener('click', function() {
  message.textContent = 'বাটন ক্লিক করা হয়েছে!';
  message.style.color = 'green';
});

HTML:

<button id="myButton">ক্লিক করুন</button>
<p id="message"></p>
